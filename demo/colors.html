Colors in SCSS can be referenced using the color(...) function. Below are a few examples.

Note: Color values and color(...) function definition can be found in styles/base/_colors.scss.

/* main */

background: color(primary);
background: color(accent);

color: color(white);
color: color(neutral);
color: color(black);


/* auxilliary */

background: color(danger);
background: color(warning);
background: color(success);
background: color(info);


/* neutrals */

background: color(neutral, 100);
background: color(neutral, 200);
border: solid 1px color(neutral, 300);
color: color(neutral, 400);
color: color(neutral, 500);
color: color(neutral, 600);
color: color(neutral, 700);


/* text */

color: color(text, dark);
color: color(text, dark, accent);
color: color(text, dark, subtle);
color: color(text, light);
color: color(text, light, accent);
color: color(text, light, subtle);
